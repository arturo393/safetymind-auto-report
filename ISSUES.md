# Known Issues

This file documents known bugs, feature requests, and planned improvements.

## Bugs

### [HIGH] Font Rendering
- **Description**: Some accented characters may not render correctly in standard PDF viewers if the host system lacks Roboto font.
- **Affected:** All Reports.

### [MEDIUM] Gantt Date Format
- **Description**: Gantt chart dates might be too small on compact layouts.
- **Affected:** Kickoff Report.
